# Gap Analysis Engine：从当前背景到目标岗位

## 核心逻辑

Gap Analysis不是列缺点，而是把“目标岗位每天要做什么”拆成能力，再对照你已有证据。真正有用的Gap Analysis必须输出优先级：哪些短板最影响入职，哪些可入职后补，哪些是长期领导力。

## 输入模板

| 输入 | 示例 |
|---|---|
| 当前背景 | 乳腺外科博士后，TNBC方向，论文/课题经验 |
| 目标岗位 | 百利天恒MA / Medical Manager |
| 目标资产 | BL-B01D1，ADC，TNBC，肺癌 |
| 不喜欢 | 基础科研、长期医院路径 |
| 优势 | 临床深度、乳腺癌患者路径、学术训练 |
| 风险 | 缺少药企语言、KOL策略、商业化和合规经验 |

## 输出：已具备能力

1. 乳腺癌临床语境：能理解TNBC治疗线、新辅助、复发转移路径。
2. 研究训练：能读论文、理解研究问题。
3. 专科可信度：与乳腺肿瘤KOL沟通有临床共同语言。
4. 医学写作基础：论文和基金经历可迁移到publication和medical communication。

## 输出：缺失能力

| 缺口 | 为什么重要 | 补齐方式 |
|---|---|---|
| ADC竞争情报 | MA必须回答SG/T-DXd/Dato-DXd/BL-B01D1如何比较 | 做横向表和KOL FAQ |
| Medical Strategy | MA不是讲文献，而是制定医学计划 | 写BL-B01D1 Medical Plan |
| Evidence Generation | 上市后要设计RWE/IIT | 写一个post-IO TNBC RWE synopsis |
| 合规边界 | off-label和医学沟通风险高 | 学SOP场景题 |
| KOL Engagement | 不是拜访，而是洞察转策略 | 模拟Ad Board议题 |
| Commercial Acumen | 医学策略需理解竞争与生命周期 | 做竞品时间线 |

## 优先补齐路径

### 第1个月

读完ADC Competitive Intelligence、TNBC Treatment Evolution、BL-B01D1 Intelligence Hub。输出一张ADC竞争格局表。

### 第2个月

完成Medical Strategy Core、KOL Engagement、Medical Communication。输出一份BL-B01D1 KOL Advisory Board问题清单。

### 第3个月

完成Evidence Generation和Lifecycle Management。输出一份post-IO TNBC RWE synopsis和一页MA面试故事。

## 转型周期

3个月：可准备MA入门面试。  
6个月：可胜任Medical Advisor/Medical Manager基础工作。  
12个月：可形成肿瘤ADC医学策略能力。  
3-5年：有机会向Medical Director路径发展。
