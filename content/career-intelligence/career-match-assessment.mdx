# Career Match Assessment：岗位匹配系统

## 使用方法

给自己每项打0-5分。0代表完全不喜欢或不擅长，5代表非常喜欢且有证据证明。不要按理想中的自己打分，要按过去三年真实行为打分。

## 评估维度

| 维度 | 说明 |
|---|---|
| Clinical Depth | 疾病、诊疗、患者路径、临床判断 |
| Trial Literacy | Protocol、终点、统计、GCP |
| Data Interpretation | PFS/OS/ORR/安全/亚组/竞品 |
| Communication | 与专家、跨部门、非医学同事沟通 |
| Strategy | 能否从数据推导下一步 |
| Detail Tolerance | 是否能处理文档、流程、数据细节 |
| Travel/Social Energy | 是否适应外勤和专家关系 |
| Risk Judgment | 是否喜欢安全性和责任判断 |
| Leadership | 是否愿意承担团队和业务结果 |

## 匹配逻辑

| 岗位 | 高匹配信号 | 低匹配信号 |
|---|---|---|
| CRP | Trial Literacy + Data + Detail高 | 不喜欢文档和流程 |
| Clinical Scientist | Trial + Strategy + Data高 | 不愿跨部门推进 |
| MA | Clinical + Communication + Strategy高 | 不喜欢合规边界和会议 |
| MSL | Communication + Travel高 | 不愿外勤或社交耗能 |
| PV | Risk + Detail高 | 不喜欢重复个案和时限压力 |
| Medical Director | Strategy + Leadership + Communication高 | 只想做个人贡献者 |

## 输出模板

```text
目标岗位：
匹配度：
优势：
短板：
优先补齐能力：
3个月行动：
6个月行动：
12个月行动：
风险提醒：
```

## 示例：乳腺外科博士后 → 百利天恒MA

匹配度：中高。  
优势：乳腺癌临床深度、TNBC患者路径、手术/新辅助经验、论文和课题训练。  
短板：药企合规、Medical Strategy、KOL engagement、ADC竞争情报、RWE/IIT设计、商业化语境。  
优先补齐：ADC横向比较、BL-B01D1策略、SG/T-DXd/Dato-DXd证据、Medical Communication、off-label边界。  
预计周期：3个月可准备面试，6-12个月形成MA能力闭环。
