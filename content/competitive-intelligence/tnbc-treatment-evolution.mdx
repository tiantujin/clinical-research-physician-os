# TNBC Treatment Evolution：从化疗时代到ADC+IO时代

## 横向时间轴

| 时代 | 核心治疗 | 代表研究 | 成功/失败逻辑 | 对未来影响 |
|---|---|---|---|---|
| 化疗时代 | 紫杉、铂类、卡培他滨等 | 多项化疗研究 | ORR有限，PFS/OS短，缺少精准人群 | 建立未满足需求 |
| PD-1/PD-L1时代 | IO+化疗 | KEYNOTE-355, KEYNOTE-522, IMpassion130 | biomarker、人群和终点决定成败 | PD-L1/CPS、pCR/EFS成为关键语言 |
| ADC时代 | TROP2/HER2 ADC | ASCENT, DESTINY-Breast04 | ADC用随机证据证明后线或HER2-low价值 | 后线到前线、检测与安全管理 |
| ADC+IO时代 | ADC联合PD-1/PD-L1 | BEGONIA, ASCENT-04, TROPION-Breast02 | 机制互补但毒性和增量价值必须证明 | 一线TNBC竞争重构 |
| Post-IO时代 | ADC、rechallenge、序贯 | ASCENT-03/04, RWE/IIT | 既往IO暴露改变疗效解释 | post-IO成为独立开发人群 |
| 未来时代 | 双靶ADC、精准序贯、MRD/RWE | BL-B01D1、HER3-DXd、Dato-DXd | biomarker和序贯决定差异化 | 医学策略比单药数据更重要 |

## 研究横向比较

| 研究 | 为什么这样设计 | 为什么成功/失败 | 未来影响 |
|---|---|---|---|
| KEYNOTE-355 | 一线mTNBC，IO+化疗，对PD-L1 CPS人群分层 | 成功关键是选择一线、联合化疗、PD-L1 CPS富集和PFS/OS终点 | 建立PD-1+化疗一线模式，强化biomarker策略 |
| KEYNOTE-522 | 早期高危TNBC，新辅助+辅助IO，pCR/EFS | 成功因为早期免疫敏感、终点与长期风险相关 | 推动早期TNBC免疫治疗标准 |
| IMpassion130 | PD-L1+一线mTNBC，atezolizumab+nab-paclitaxel | 反转在于伴随诊断、确认研究和地区监管差异 | 提醒MA不能只看初始阳性，要看确证证据 |
| ASCENT | 后线mTNBC，SG vs TPC | 成功因为未满足需求明确、随机对照、PFS/OS一致 | TROP2 ADC成为后线标杆 |
| ASCENT-03 | SG更早线/特定人群探索 | 设计关键是对照是否随标准变化更新 | 影响SG前移和post-IO定位 |
| ASCENT-04 | SG联合IO一线策略 | 成败取决于ADC+IO增量价值、毒性、PD-L1人群 | 决定TROP2 ADC是否进入一线 |
| DESTINY-Breast04 | HER2-low定义人群，T-DXd vs TPC | 成功因为机制、检测和随机生存获益结合 | 重塑HER2-low治疗格局 |
| BEGONIA | 一线TNBC平台探索ADC/IO组合 | 价值在信号筛选，不是注册结论 | 训练如何从小样本ORR走向确认研究 |
| TROPION-Breast01 | Dato-DXd在HR+/HER2-后线 | 价值在TROP2 ADC跨乳腺癌亚型竞争 | 为TNBC布局提供安全和差异化线索 |
| PANKU-Breast02 | 中国TNBC/ADC相关研究场景 | 应关注中国路径、样本、对照和监管外推 | 本土研究可补中国真实世界与注册策略 |

## 为什么不能只讲结果

同样是ORR提高，含义完全不同：如果在后线无药可用人群且DoR长，可能支持加速路径；如果在一线已有强对照且PFS/OS无改善，可能只是短暂缩瘤。TNBC策略必须先看治疗线、对照、biomarker、终点层级、安全和竞品时间线，再看数据。

## 成功案例、失败案例、反转案例

| 类型 | 案例 | 经验 |
|---|---|---|
| 成功 | KEYNOTE-522 | 早期高危场景免疫治疗有更大治愈潜力 |
| 成功 | KEYNOTE-355 | biomarker富集决定IO一线成败 |
| 成功 | ASCENT | ADC注册价值需要随机PFS/OS |
| 成功 | DESTINY-Breast04 | 检测定义可创造新治疗人群 |
| 失败/受挫 | IMpassion130路径反转 | confirmatory evidence与诊断一致性关键 |
| 失败 | ORR高PFS不显著ADC组合 | 缩瘤不等于临床获益 |
| 失败 | ADC+IO毒性高停药多 | 联合策略必须证明净获益 |
| 反转 | HER2-low从“阴性”到可治疗 | 病理教育成为医学策略 |
| 反转 | Post-IO成为独立人群 | 既往治疗暴露改变开发逻辑 |
| 反转 | ADC序贯问题上升 | 单药上市后真正问题才出现 |

## Medical Affairs Application

如果我是MA：我会把TNBC治疗演进讲成“为什么治疗线和biomarker决定用药”，而不是讲药物清单。  
如果我是Medical Manager：我会用这张演进图设计KOL Advisory Board，问题聚焦post-IO、ADC序贯、HER2-low检测和一线ADC+IO。  
如果我是Medical Director：我会决定哪些证据缺口必须由RWE/IIT补齐，哪些话题不能越过标签边界。
